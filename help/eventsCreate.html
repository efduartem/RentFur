Crear Evento

Esta vista permite registrar un nuevo evento completando los siguientes datos principales (obligatorios):

  1. Fecha del Evento
  2. Estado (PRESUPUESTADO o CONFIRMADO)
  3. Cliente
  4. Lugar de entrega
  5. Mobiliarios

Acciones

    Seleccionar Cliente: muestra una ventana de dialogo que permite buscar un cliente y seleccionarlo


    Agregar Mobiliario: muestra una ventana de dialogo que permite buscar mobiliarios y seleccionar varios de ellos para agregarlos a la tabla de detalles


Tabla de detalles

  1. Se muestran los datos de los mobiliarios agregados
  2. La columna "Disponibilidad" muestra la cantidad disponible del mobiliario en la fecha seleccionada
  3. La columna "Cantidad" permite ingresar la cantidad a ser contratada de cada mobiliario
  4. (*) Es obligatorio para finalizar el registro completar las cantidades de todos los mobiliarios agregados
  5. Con el ingreso de las cantidades respetivas, en la parte superior derecha de la evento es actualizado el total a ser abonado por el evento

    Crear Evento: Guarda en la base de datos del sistema los datos del evento.

Si el estado fue actualizado a "CONFIRMADO", el mismo compromete el stock de mobiliarios para la fecha del evento.

    Cancelar: son descartados todos los datos y se vuelve al "Calendario de Eventos"
