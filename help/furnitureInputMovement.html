Registro de Entradas al Inventario

Esta vista permite registrar un nuevo movimiento de entrada para mobiliario completando los siguientes datos.

  • Fecha

  • Concepto

    • ENTRADA POR AJUSTE DE INVENTARIO

    • INVENTARIO INICIAL (El inventario inicial se realiza por unica vez, por cada mobiliario)

  • Mobiliarios

Con el boton    Seleccionar Mobiliarios, se muestra una ventana que permite buscar y seleccionar uno o varios mobiliarios


Para registrar el movimiento se debe presionar el boton    Guardar, en caso contrario presionar    Cancelar
