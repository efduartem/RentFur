Registrar Factura

Esta vista permite registrar una nueva factura completando los siguientes datos principales (obligatorios):

  1. Detalles de Factura (Items)

Acciones

Dependiendo del tipo de facturación se pueden:

  1. Detallada:

       Seleccionar Detalles: Se debe seleccionar uno por uno cada mobiliario contratado (aún no facturado)

  2. Por Concepto:

       Agregar Detalles: agrega una linea a la tabla de detalles de factura

Tabla de Detalles

  1. Si se trata de facturación por concepto, se debe ingresar la descripcion para el item
  2. Si se trata de facturación por concepto, se debe ingresar la cantidad para el item
  3. Si se trata de facturación por concepto, se debe ingresar el precio unitario para el item

    Guardar : Guarda en la base de datos del sistema la factura.

   Cancelar: son descartados todos los datos y se vuelve al "Detalles de Evento"
