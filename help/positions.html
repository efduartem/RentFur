Cargos

Busqueda de Cargos

En esta vista se pueden filtrar los cargos creados por los campos:

  • Codigo

  • Descripcion

  • Permiso

Para obtener los resultados de dicho filtro se debe presionar sobre el boton    Buscar

Debajo se muestra la tabla con la lista de cargos que corresponde a la busqueda realizada, la misma muestra los datos principales de cada cargo como:

  • Codigo
  • Descripcion
  • Permisos
  • El botón "Ver": Permite ingresar a la vista donde se ven los detalles del cargo y poder editarlo.


Detalles de Cargo

En esta vista se muestran los datos de un cargo seleccionado:

  • Descripcion

  • Permisos

Estos datos seran habilitados para modificar presionando el boton     Editar

Selección de Permisos


  • Los permisos brindan, cada uno, accesos a las opciones principales del sistema

  • Por cada permiso otorgado, es posible especificar que dicho acceso será valido para realizar consulta de datos en dicha opción.

    Esto siginifica que no se habilitaran opciones que impliquen registro(creación) y/o modificación de datos

Para confirmar el cambio se debe presionar el boton    Guardar

En caso contrario presionar    Cancelar


  

Crear Cargo

Esta vista permite registrar un nuevo cargo

Los datos necesarios son:

  1. Descripcion
  2. Permisos

Selección de Permisos


  • Los permisos brindan, cada uno, accesos a las opciones principales del sistema

  • Por cada permiso otorgado, es posible especificar que dicho acceso será valido para realizar consulta de datos en dicha opción.

    Esto siginifica que no se habilitaran opciones que impliquen registro(creación) y/o modificación de datos


    Crear : Guarda en la base de datos del sistema el nuevo cargo.


    Cancelar: son descartados todos los datos y se vuelve a la "Busqueda de Cargos"
