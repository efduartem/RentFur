Sistema


  • Cambiar contraseña: permite al usuario logueado, cambiar su contraseña de acceso al sistema. Para ello debe proporcionar los siguiente datos:

    • Contraseña Actual
    • Nueva Contraseña
    • Confirmar Contraseña


        Guardar: Actualiza en la base de datos del sistema la nueva contraseña del usuario.

        Cancelar: son descartados todos los datos


  • Ayuda: Permite acceder a este menu de ayuda
  • Salir: Cierra la sesión y se sale del sistema cerrando todas las ventanas
