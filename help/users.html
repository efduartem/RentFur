Usuarios

Busqueda de Usuarios

En esta vista se pueden filtrar los usuarios por los campos:

  • Codigo

  • Nombre de Usuario

  • Nombre Completo

  • Cargo

  • Estado

Para obtener los resultados de dicho filtro se debe presionar sobre el boton    Buscar

Debajo se muestra la tabla con la lista de usuarios que corresponde a la busqueda realizada, la misma muestra los datos principales de cada usuario y se cuenta con la posibilidad de:

  • Activar/Inactivar
  • Ver: Ingresar a la vista donde se ven los detalles del usuario.


Detalles de Usuario

En esta vista se muestran los datos de un usuario seleccionado:

  • Codigo

  • Nombre Usuario

  • Nombre Completo

  • Cargo

  • Estado

  • Contraseña

Estos datos seran habilitados para modificar presionando el boton     Editar

Para confirmar el cambio se debe presionar el boton    Guardar

En caso contrario presionar    Cancelar


  

Crear Usuario

Premite registrar un nuevo usuario completando los siguientes datos:

  • Nombre Usuario

  • Nombre Completo

  • Contraseña

  • Confirmar contraseña

  • Cargo

    Crear: Guarda en la base de datos del sistema los datos del nuevo usuario.

    Cancelar: son descartados todos los datos y se vuelve a la "Busqueda de Usuarios"
